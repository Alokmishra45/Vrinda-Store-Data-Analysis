## Vrinda-Store-Data-Analysis 📊
Sales data analysis using Excel to find insights and business trends

## Project Overview

This project analyzes the sales data of Vrinda Store using Microsoft Excel to identify business insights, sales trends, and customer purchasing behavior.

The objective of this project is to transform raw data into meaningful insights that help in business decision-making.

---

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Cleaning
* Dashboard Creation

---

## Key Steps Performed

### 1. Data Cleaning

* Removed duplicate values
* Handled missing data
* Standardized column formats

### 2. Data Analysis

* Created Pivot Tables
* Analyzed sales by category, gender, and location
* Identified top performing products

### 3. Dashboard Creation

An interactive dashboard was built to visualize:

* Total Sales
* Orders by Gender
* Orders by State
* Sales by Category
* Monthly Sales Trend

---

## Key Insights

* Women customers contributed higher sales than men.
* Certain states generated the majority of the revenue.
* Seasonal trends were observed in monthly sales.
